Accessible Parse Tree Renderer (Bracket Notation)

Enter a tree like (S (NP (A the) (N dog)) (VP (V loves) (NP (A the) (N cat)))) and render it as an image.

Source: Based on jsSyntaxTree by André Eisenbach et al. (GPL-2.0), with accessibility enhancements.

Input

Format: parentheses for nodes. A node is (Label child1 child2 ...). Leaves can be plain tokens, or (Label token).
Example
Try: 
(S
  (NP (A the) (N dog))
  (VP (V loves) (NP (A the) (N cat))))

Output

Non-terminal (square) Terminal (circle)

Nothing rendered yet. Enter a tree and select “Render”.

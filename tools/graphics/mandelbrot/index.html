Mandelbrot Prep: Complex Numbers & One Iteration

This page gives the minimum complex-number background needed for a Mandelbrot renderer, followed by an interactive plane that shows both (1) complex numbers as points and (2) one iteration of f(z) = z² + c.

Minimal notes you need for the Mandelbrot set

1) Complex numbers

A complex number is written as z = a + bi, where a and b are real numbers and i is the imaginary unit with i² = −1.

You can view z = a + bi as the 2D point (a, b). The horizontal axis is the real part, and the vertical axis is the imaginary part.

2) Magnitude (distance from the origin)

The magnitude of z = a + bi is: 

|z| = √(a² + b²)

3) Multiplication (you’ll need this for squaring)

Multiply using distributivity and i² = −1: 

(a+bi)(c+di) = (ac − bd) + (ad + bc)i

In particular, squaring z = a + bi gives: 

z² = (a+bi)² = (a² − b²) + (2ab)i

4) Mandelbrot iteration

For a chosen complex number c, define a sequence: 

z₀ = 0
zₙ₊₁ = zₙ² + c

5) The escape test (what you implement in code)

If at any step you get |zₙ| > 2, then the sequence will diverge (escape to infinity). When rendering, you iterate up to a max number of steps and color by how quickly the orbit escapes.

Optional: code-friendly form (no complex type needed)

If you store a complex number as (a, b), then one iteration of z ← z² + c can be done with real arithmetic: 

Given z = (a, b) and c = (cr, ci):

a' = a² − b² + cr
b' = 2ab + ci

Interactive: points in the complex plane and one iteration

Drag the points, or focus the canvas and use arrow keys. This demo shows the mapping f(z) = z² + c for your chosen z and c.

Complex plane

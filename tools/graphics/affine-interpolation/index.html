Affine combinations on a line segment

Drag P and Q. Use the slider for t to move X where X(t) = (1−t)P + tQ. Turn on “Extrapolate” to allow t outside [0,1].

t = 0.500
Keyboard: use ←/→ to adjust t.
If enabled, t ranges from −1 to 2. Values t<0 go past P; t>1 go past Q.
Coordinates are shown in the side panel and next to points in the diagram.
Keyboard access for points
Select a point, then nudge it with Shift + arrow keys.
Drag points with the mouse or trackpad. Click a point to “select” it; then nudge with Shift + ←/→/↑/↓.
Key idea

An affine combination of points uses weights that sum to 1. Here the weights are (1−t) and t, and (1−t)+t=1.

Formulas (equivalent)
Endpoints: t=0 ⇒ X=P, t=1 ⇒ X=Q. Midpoint: t=1/2 ⇒ X=(P+Q)/2.
Weights
wP = 1 − t = 0.500 wQ = t = 0.500 wP + wQ = 1.000
Weight bars
Left bar corresponds to wP; right bar corresponds to wQ. (When extrapolating, weights can go outside [0,1]; bars are clamped for display.)
Same weights, different range: color

If X(t) = (1−t)P + tQ, then we can also set Color(X) = (1−t)Color(P) + tColor(Q). With Color(P)=black and Color(Q)=white, grayscale(X)=t.
